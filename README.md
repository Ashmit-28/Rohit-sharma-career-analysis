🏏 Rohit Sharma Career Analysis Dashboard using Power BI

📌 Project Overview

This project focuses on analyzing the career performance of Rohit Sharma, one of India's leading international cricketers, using Power BI. The dashboard was developed to visualize batting performance across different cricket formats including ODI, T20, and Test matches.

The main objective of this project is to transform cricket performance data into meaningful visual insights, enabling analysis of batting consistency, performance trends, and format-wise achievements throughout Rohit Sharma's career.

🎯 Objectives
Analyze Rohit Sharma's batting performance across ODI, T20, and Test formats.
Track key performance metrics such as total runs, batting average, strike rate, and centuries.
Compare performance across different match formats.
Identify performance trends over time.
Create an interactive dashboard for exploring cricket statistics visually.


🛠️ Tools & Technologies Used
Power BI – Data visualization and dashboard development
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – Calculation of cricket performance metrics
Excel / Cricket Dataset – Source of batting statistics
Data Visualization Techniques – Charts, KPI cards, and filters


📂 Dataset Description

The dataset contains detailed batting statistics of Rohit Sharma across different international cricket formats.

Key Columns Included:

Runs Scored
Balls Faced
Strike Rate
Batting Average
Centuries (100s)
Half-Centuries (50s)
Opponent Team
Match Year
Venue
Dismissal Type

The dataset was structured to enable comparison of performance across formats and time periods.

The following data preparation steps were performed using Power Query:

📈 Key Insights Generated

Some important insights derived from the dashboard include:

Rohit Sharma showed strong performance consistency in ODI format compared to other formats.
Significant growth in total runs and strike rate was observed over the years.
Certain opponent teams showed higher scoring performance patterns.
Century frequency increased during peak career years.
Format-wise performance trends highlighted variations in scoring style.

These insights help demonstrate how sports analytics can be used to evaluate player performance and career progression.
