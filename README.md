# Rohit-sharma-career-analysis

Developed a sports analytics dashboard using Power BI to analyze batting performance across ODI, T20, and Test formats.
Cleaned and transformed cricket data using Power Query to prepare performance datasets.
Created DAX measures to calculate key statistics such as Runs, Batting Average, Strike Rate, and Centuries.
Designed interactive visualizations and filters to compare performance across formats, opponents, and years.
Identified career performance trends and highlighted peak performance periods.
